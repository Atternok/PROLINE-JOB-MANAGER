# PROLINE INVOICE MANAGER
## Running the project locally
Follow the steps below to run PROLINE-JOB-MANAGER on your local machine.

1. Clone the repository
```bash
git clone https://github.com/Atternok/PROLINE-JOB-MANAGER.git
```

2. Navigate to the project directory
```bash
cd PROLINE-JOB-MANAGER
```

3. Set up Node.js (required)
This project requires Node.js v20 or higher.
```bash
nvm install 20
nvm use 20
node -v
```
Ensure the output shows v20.x.x (or higher).

4. Install dependencies
```bash
npm install --include=optional
```
Verify that Tailwind CSS is installed:
```bash
npm list tailwindcss
```

5. Start the development server
```bash
npm run dev
```

6. Open the app
Visit the following URL in your browser:
`
http://localhost:3000
`
